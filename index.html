User Guide
  1. Select the appropriate tab for what you want to report (User, Bot, or Group)
  2. Enter the username or ID of the account you're reporting
  3. Select the reason for your report from the dropdown menu
  4. Provide any additional details that might help with the investigation
  5. Optionally modify the API endpoint if you're using a custom server
  6. Click "Submit Report" to send your report
API Documentation
API Endpoints:
POST /v1/report/user
POST /v1/report/bot
POST /v1/report/group
Example Request (User Report):
{
    "type": "user",
    "identifier": "@username_or_phone",
    "reason": "spam",
    "details": "This user is sending spam messages",
    "block": true
}
Example Response:
{
    "success": true,
    "message": "User successfully reported",
    "report_id": "USER-1689345678901",
    "timestamp": "2025-08-14T12:34:56Z"
}
